发送请求

1. urllib

包括四个模块：

  • request:
  • error
  • parse
  • robotparser

request

  • ==urlopen()==

    • 参数列表
    • url
    • data
    • timeout

  • ==Request()==

    • 参数列表
    • url
    • headers
    • origin_req_host
    • method

  • ==Handler,Opener==

error

  • URLError
  • HTTPError

parse

  • urlparse
  • urlunparse
  • urlencode
  • parse_qs
  • quote
  • unquote

2. requests

  • get

    • 二进制数据抓取

  • post

  • 文件上传
  • 获取cookie
  • 设置cookie
  • 维持会话
  • ssl证书验证
  • 设置代理
  • 超时设置
  • 身份认证
  • Prepared Request

Xpath

语法

表达式 描述
nodename 选取此节点的所有子节点
/ 从根节点选取（取子节点）
// 从匹配选择的当前节点选择文档中的节点，而不考虑它们的位置（取子孙节点）
. 选取当前节点
.. 选取当前节点的父节点
@ 选取属性
